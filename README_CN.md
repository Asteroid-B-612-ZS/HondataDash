# HondataDash

Hondata FlashPro 蓝牙仪表盘，专为 800×480 横屏车机设计。实时显示调校最常用的参数，通过颜色直观区分安全、警告和危险区间。

## 这个 App 是什么

- 通过蓝牙连接 Hondata FlashPro，实时读取发动机数据
- 根据工况自动过滤：滑行断油（DFCO）、暖机、同步恢复时的数据不会被误当成稳定调校值
- 零依赖，纯 Android 原生开发，APK 只有 55 KB

## 这个 App 不是什么

- 不是通用 OBD 读取器，只支持 FlashPro 蓝牙协议
- 不能替代 FlashPro Manager，不能刷写或修改标定
- 不是自动调校工具，只负责显示，调校判断靠你自己

### 转速灯条

- 6 颗 LED，随转速递进点亮
- 阈值（针对 1.5T 调校）：3000 / 3500 / 4000 / 4500 / 5000 / 5500
- 颜色：绿×2 → 黄×2 → 红×2
- 超过 6400 转全亮并以 10Hz 闪烁

### 主数据颜色（V2.7.0 语义颜色）

主数值根据数值范围自动变色，一眼区分安全与异常：

| 卡片 | 绿色（安全）| 黄色（警告）| 红色（危险）| 其他 |
|------|------------|------------|------------|------|
| ECT 水温 | 80~95°C | 96~100°C | >100°C 闪烁 | <80°C 蓝色（冷车）|
| L.TRIM 长期修正 | ±5% 以内 | ±15% 以内 | 超过 ±15% | — |
| S.TRIM 短期修正 | ±5% 以内 | ±15% 以内 | 超过 ±15% | — |
| IGN 点火角 | 0° 及以上 | -5°~0° | 低于 -5° | — |
| MAP 相对增压 | 1.45bar 以内 | 1.45~1.60bar | 超过 1.60bar | 负值蓝色（真空）|
| A/F 空燃比 | 14.5~15.5 | 11~14.5 / 15.5~18 | <11 / WOT 过稀闪烁 | 断油时显示 DFCO |

刻度条保持原有的方向/情绪模式，不随主文字变色。

### 底部数据栏

| 项目 | 含义 |
|------|------|
| K.C | 爆震控制（<55 绿, 55~65 黄, >65 红闪）|
| C1~C4 | 各缸爆震计数（0 绿, 1 黄, ≥2 红）|
| K.R / K.L | 爆震退角 / 爆震上限 |
| BAT | 电瓶电压 |
| F.P | 油压 |
| W.G | 废气旁通 |
| T.P | 节气门开度 |

## 数据来源

- **蓝牙**：连接 FlashPro，50Hz 轮询。MAC 地址写在 `BluetoothSource` 里
- **Demo 模式**：用真实 LOG 数据回放，不需要接车

## 工况过滤

不是所有读数都能信。以下工况的数据会被特殊处理：

- **DFCO 断油滑行**：A/F、IGN、S.TRIM 显示 "DFCO"，不记录极值
- **暖机**：水温没上来时，相关参数灰显，表示参考价值低
- **同步恢复**：断油后刚恢复喷油时，短暂显示 "SYNC"，等数据稳定
- **低置信度**：动态工况（换挡、突变）下读数不稳时灰显

WOT 判定不看节气门开度，而是看 ECU 策略信号（开环 + 浓油 + 增压 + 转速），这对扭矩模型的 L15B7 更准确。

## 极值记录

- **过滤规则**：断油、暖机、瞬态尖峰不进入 MAX/MIN 记录
- **两种极值**：本次行驶全程极值和近期动态极值分开
- **基准覆盖**：发动机稳定运转后，用当前真实值作为极值起点，不从噪声起步
- **断线保护**：短时蓝牙断连（5 分钟内）不丢极值；长断后熄火才重新开始

## 蓝牙断线恢复

- 断线后自动重连，界面不退出
- 切后台再回来不会误判断线（V2.6.9 修复）
- 蓝牙被关掉时停止重连，不会狂连耗电
- 重连失败指数退避：1 秒 → 2 秒 → 4 秒 → 8 秒

## 安装

GitHub Release 不提供 APK（正式版包含私有设备 MAC）。自己编译——见下面。

## 编译

```bash
JAVA_HOME="~/.jdks/jbr-17.0.14" ./gradlew assembleRelease
adb install -r app/build/outputs/apk/release/app-release.apk
```

需要 Demo 版（LOG 回放，不接车），把 `MainActivity` 里 `USE_DEMO` 改成 `true` 再编译。

## 限制

- 固定 800×480 布局，不适配其他屏幕
- 只支持 FlashPro 蓝牙协议
- MAC 地址硬编码，换 FlashPro 要改代码
- 没有设置界面，阈值都是代码常量
